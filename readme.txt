Juassi 2.0.1

To install:
1) Upload to web hosting
2) Check .htaccess if you're not installing Juassi in the root folder
3) go to /juassi-install/ in your web browser and follow the instructions

**For more information on how to install please check the docs folder.**

Requirements:
PHP 5.2 required (with PHP PDO class).
MySQL 4.1+ or SQLite.
Mod_Rewrite highly recommended.

If you have any problems sign up to the mailing list(s) and ask a question (found here: http://www.juassi.com/archive/2006/10/05/juassi-mailing-lists/).
